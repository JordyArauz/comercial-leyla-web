# Comercial Leyla Web

Página web estática tipo catálogo para el proyecto multimedia **Comercial Leyla: Todo para tu Hogar**.

## Qué incluye

- `index.html`
- `styles.css`
- `script.js`
- Logo en `assets/img/logo.png`
- 10 imágenes de productos de muestra
- Video placeholder en `assets/video/spot-comercial-leyla.mp4`
- Botones de WhatsApp funcionales
- Animaciones CSS
- Interacciones con JavaScript
- Canvas decorativo
- Diseño responsive y mobile first

## Cómo probar

Abre el archivo `index.html` directamente en el navegador.

## Cómo cambiar productos

Edita el arreglo `products` dentro de `script.js`.

Ejemplo:

```js
{
  name: "Olla grande",
  category: "Cocina",
  image: "assets/img/producto-01.jpg",
  description: "Descripción del producto."
}
```

## Cómo reemplazar imágenes

Cambia los archivos:

- `assets/img/producto-01.jpg`
- `assets/img/producto-02.jpg`
- hasta `assets/img/producto-10.jpg`

por fotos reales de la tienda, manteniendo los mismos nombres.

## Cómo reemplazar el spot

Cambia:

`assets/video/spot-comercial-leyla.mp4`

por el video final editado en CapCut.

## Cómo subir a Vercel

1. Crea un repositorio en GitHub.
2. Sube toda esta carpeta.
3. Entra a Vercel.
4. Importa el repositorio.
5. Framework: Other.
6. Build Command: dejar vacío.
7. Output Directory: dejar vacío o usar `./`.
8. Deploy.
